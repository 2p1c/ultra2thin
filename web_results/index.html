Summary

Architecture

Input:  32×32=1024 measurement points × 256 time samples
      Source position (xs, ys)

Encoder:
  SignalEncoder (1D CNN):  raw waveform → 48D feature per point
  GeometryEncoder (Fourier): position + source-relative geometry → 48D feature

Decoder:
  feature sum → reshape to (48, 32, 32) 2D map → ThinUNet → thickness map (32×32)

Training:  500 unique random plates × 1 configuration each
  Phase 1: OneCycleLR 2e-3 → 0 (25 epochs)
  Phase 2: CosineAnnealing 2e-4 → 0 (15 epochs)
  Loss: MSE(predicted_map, ground_truth_map)
  Device: CPU  |  515K parameters

Physics: A0 Lamb wave dispersion — c_p(f,d) varies strongly with thickness.
  Phase velocity encoded per frequency bin (Rayleigh-Lamb equation solved numerically).
  ~60× more thickness-sensitive than through-thickness pulse-echo.
