📐 新架构 vs 旧架构

v1 (旧):
  固定16接收器 4×4 → 整张32×32厚度图
  源位置固定 · 点阵大小固定
  ✗ 采集方式受限

v2 (新):
  N个任意位置测量点 → N个逐点厚度
  → 空间插值 → 2D厚度图
  源位置可变 · 点阵大小可变 (9~64点)
  ✓ 灵活采集

逐点编码:
  signal → CNN → 信号特征
  pos → Fourier(绝对位置)
  pos-src → Fourier(相对源向量)
  ||pos-src|| → Fourier(距离)
  angle → sin/cos(方向)
  → 拼接投影 → 128D 逐点特征

🔬 编解码信息流

每个测量点编码 (GeometryEncoder):
  FourierFeatures(绝对位置, 12频) → 48D
  FourierFeatures(相对源, 12频) → 48D
  FourierFeatures(距离, 8频) → 16D
  sin(角度), cos(角度) → 2D
  ─────────────────────────
  拼接 → Linear→LN→ReLU→Linear → 128D

点间信息共享:
  3层 Transformer (4头)
  各点互相查询空间上下文

2D可视化:
  逐点预测 → cubic插值 → 32×32厚度图
